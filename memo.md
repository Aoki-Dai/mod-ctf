## SquashFS

SquashFSは、Linux向けの軽量で高圧縮な読み込み専用（Read-only）ファイルシステムです。ファイル、inode、ディレクトリを効率的に圧縮し、OSのライブイメージ（LiveCD/USB）、組み込みLinuxのルートファイルシステム、アーカイブなどで広く利用されています。

## Ghidra

Ghidraは、アメリカ国家安全保障局によって開発されたオープンソースのリバースエンジニアリングツール

## 多表式換字暗号

多表式換字暗号（Polyalphabetic Substitution Cipher）は、複数の換字表（アルファベット配列）を周期的に切り替えて文字を置換する古典暗号です。単一の換字表を使う単一換字式暗号の弱点である頻度分析を無効化し、ヴィジュネル暗号がその代表例として知られています。

## tshark

TSharkは、人気のGUIパケットキャプチャツール「Wireshark」のコマンドライン版（CUIツール）であり、ネットワークトラフィックのキャプチャ、解析、デコードをコマンド上で高速に行えるツールです。GUI環境がないサーバーでの使用、スクリプトによる自動化、特定のフィールド抽出などに最適で、Wiresharkがインストールされている環境であればWindows、Linux、Macで利用可能です。

## objdump

objdump は、バイナリファイル（実行ファイルやオブジェクトファイル）の内部情報を表示するためのコマンドラインツールです。GNU Binutils の一部として提供されており、プログラムの動作解析やデバッグ、CTF（セキュリティ競技）などで広く利用されます。
→逆アセンブリに使用

```bash
# 逆アセンブル結果を表示
objdump -d a.out
```

## tcpdump

tcpdumpは、ネットワーク上を流れるパケットをキャプチャし、解析するための強力なオープンソース・コマンドラインツールです。Linux、macOS、Unix系OSで動作し、通信トラブルシューティングやデバッグ、セキュリティ監視に不可欠です。リアルタイム監視に加え、フィルタリング、PCAPファイルへの保存、Wiresharkでの詳細分析が可能です。

## CTFにおいてJohn

### セキュリティ研究者：John Hammond氏

John Hammond氏は、世界的に有名なサイバーセキュリティ研究者であり、教育者です。

### パスワード解析ツール：John the Ripper

CTFの競技中に使用される、有名なパスワードクラッキング（解析）ツールです。
用途: 暗号化されたパスワード（ハッシュ値）を高速で解析し、元の文字列を特定するために使われます。
コマンド: CTFの問題では、よく「zip2john」でZIPファイルのハッシュを抽出し、「john」コマンドで辞書攻撃を仕掛けるといった手順が使われます。
出題例: 国内のksnctfなど、問題タイトル自体が「John」となっているものは、このツールの使用を想定した問題であることが多いです。

## Aircrack-ng

Aircrack-ngは、802.11無線LANのセキュリティを評価するためのツールセットであり、パケットスニッフィング、WEP/WPA/WPA2-PSKの暗号解読（クラック）、分析、テスト機能を提供します。

Kali LinuxやParrot OSなどのセキュリティ特化型ディストリビューションにプリインストールされています。

## nm

nmコマンドは、LinuxなどのUNIX系OSにおいて、オブジェクトファイル、静的ライブラリ、実行可能ファイル（ELF形式など）のシンボルテーブル（関数名や変数名）を表示するデバッグツール

使用例：オブジェクトファイルのシンボルを確認する

```bash
nm test.o
```

## Tor Browser

- Tor Browserは、オンラインでの匿名性とプライバシーを最大限に高めるために設計された無料のオープンソースWebブラウザです。Firefoxをベースにしており、Windows、macOS、Linux、Androidで利用可能

- オニオンルーティング: 通信データを「玉ねぎ」の層のように多重に暗号化し、世界中のボランティアが運営する3つの中継サーバー（ノード）を経由させます。
  - ガードリレー: 最初の入り口。ユーザーのIPアドレスを知っていますが、中身は見えません。
  - 中間リレー: 通信を中継。送信元も最終目的地もわかりません。
  - 出口リレー: 最後の出口。データはここで復号され、目的地に届きます。
- .onionサイト: 通常のブラウザではアクセスできない「ダークウェブ」のサイトを閲覧できます。

```sh
brew install --cask tor-browser
```

## TestDisk

- 「TestDisk」は、失われたパーティションの修復や、起動しなくなったディスクの復旧に特化した、無料かつオープンソースの強力なデータ復旧ツール
- ファイル単体の復旧（画像や動画など）には、同梱されている姉妹ツールの PhotoRec を使うのが一般的

## zsteg

- PNGやBMP形式の画像ファイルに隠されたデータ（ステガノグラフィ）を検出・抽出するためのRuby製コマンドラインツール

## e2tools

- Linuxのext2/ext3/ext4ファイルシステム内のファイルを、マウントすることなく直接読み書き・操作するためのシンプルなコマンドラインツールセット

## radare2

- radare2（通称 r2）は、オープンソースの強力なリバースエンジニアリング・フレームワーク
- バイナリファイルの解析、逆アセンブル、デバッグ、パッチ当てなど、低レイヤーの多岐にわたるタスクをコマンドラインから実行

## chmod

- chmod（change mode）は、LinuxやUnix系OSでファイルやディレクトリの権限（パーミッション）を変更するためのコマンド
- 8進数または、記号で指定する。

## xxd

- xxdは、ファイルや標準入力の内容を16進数（ヘキサデシマル）でダンプ、あるいはその逆に16進数からバイナリへ復元するためのコマンド

## grep

- grep（グレップ）は、UNIX系OSの標準コマンドで、テキストファイルや標準入力から指定した文字列（パターン）に一致する行を検索して表示するツール
- 正規表現を使って複雑なパターン検索も可能

## nc

- nc（netcat）コマンドは、ネットワーク経由でTCPやUDPのデータを読み書きするための多機能ツール

## Pythonライブラリ：Z3 Solver（Z3Py）

- PythonでZ3 Solver（Z3Py）を使用すると、複雑な論理パズル、数学の問題、制約充足問題を効率的に解くことができます

## CoAPプロトコル

- CoAP（Constrained Application Protocol）は、IETFによってRFC 7252として標準化された、リソースの限られた低消費電力IoTデバイス向けの軽量なUDPベースの通信プロトコル
- HTTPのようなRESTfulな通信（GET, POST, PUT, DELETE）を、UDP上で動作させることで、高速かつ低オーバーヘッドなM2M/IoT通信を実現
- クライアント・サーバーモデル
- MQTTはPub/Sub（パブリッシュ/サブスクライブ）モデルを採用している一方、CoAPはWebベースの要求/応答（Request/Response）モデルを採用
- libcoap (coap-client): C言語で実装されたコマンドラインツール

## solve.py

- CTF（Capture The Flag）において、solve.pyは暗号解読、自動化、パズル解決のためにPythonで書かれたスクリプトファイルを指す典型的な名称

## QEMU（Quick Emulator）

- QEMU（Quick Emulator）は、異なるCPUアーキテクチャのソフトウェアを実行したり、仮想ハードウェア環境を構築したりできるオープンソースのCPUエミュレータ兼バーチャライザ

## qemu-x86_64

- qemu-x86_64は、オープンソースのエミュレータ「QEMU」に含まれるコマンドの一つ
- 主にLinuxのx86_64（64ビット）バイナリを他のアーキテクチャ上で直接実行するために使用されます。

## レールフェンス暗号

- レールフェンス暗号は、文章の文字を並べ替えることで暗号化する転置式暗号の一つ
- レールフェンスはレール（横木）を使った柵の意味で、レールにジグザクに文字を配置して最後にレールの単位で文字を繋げることで暗号化
- レールの本数が暗号のキー

## ヴィジュネル暗号

- ヴィジュネル暗号は、換字式暗号の一種で、換字表（ヴィジュネル暗号表）を用いて暗号化を行う

## スキュタレー暗号

- スキュタレー暗号は、文章の文字を並べ替えることで暗号化する転置式暗号の一つ
- スキュタレーはギリシャ語でバトンを意味する
- 棒に細長い羊皮紙を巻きつけ、羊皮紙を横断して文字を書いていくことで暗号化
- 棒の１周に書き込める文字数が暗号のキーになります。または、１行に書き込める文字数（羊皮紙を巻きつけた回数）も指定可能
